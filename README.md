## install & run

* install node.js
* `npm config set registry https://registry.npm.taobao.org`
* `npm install` to install dependencies
* `npm run start` to preview in browswer.

## publish

* npm config set registry https://registry.npmjs.org/
* npm login
* npm publish --access public
